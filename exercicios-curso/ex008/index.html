Italico/enfase

Nessa frase temos um termo em italico utilizando a tag <i> (não semantica)

Nessa frase temos um termo em italico utilizando a tag <em> (semantica)

Texto marcado

Podemos criar uma frase com o texto marcado utilizando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto peqeueno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas desconsiderado, utilizando a tag <del>

Texto incluido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois, utilizando a tag <ins>

Existe tambem o sublinhado com a tag <u> (não semantica)

Texto sobrescrito

Para inserir coisas do tipo 2x12 utilizamos a tag <sup>
